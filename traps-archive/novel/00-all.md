# 小说域陷阱

> 由小说审稿持续积累。写作/审稿时参考。

---

## AI 痕迹 / 套路化表达

| # | 陷阱 | 避免方式 |
|---|------|----------|
| 1 | 滥用"首先/其次/最后"机械罗列 | 用自然的叙事过渡，不用列举式连接词 |
| 2 | 堆砌华丽辞藻、无实质内容的空泛描写 | 用具体细节替代抽象形容词 |
| 3 | 车轱辘话——同一意思反复表达 | 删除冗余，每句话承载新信息 |
| 4 | "就在这时""突然之间""只见"等俗套过渡 | 用动作/环境变化自然过渡 |
| 5 | 万能结尾——每章都是"预知后事如何" | 用具体悬念钩子替代套话 |
| 6 | 人物情感表达单一，缺乏层次感 | 通过行为、微表情、内心独白多维展现 |
| 7 | 对话过于直白或书面化，不符合人物身份 | 根据角色性格调整对话风格 |
| 8 | "眼中闪过一丝"——AI 高频表情描写 | 用具体的肢体语言替代 |
| 9 | "嘴角勾起一抹"——AI 高频微笑描写 | 用更自然的表情变化 |
| 10 | "深吸一口气"——AI 高频情绪前奏 | 直接写人物行动或心理 |
| 11 | 三段式法则——连续三个"既…又…还…" | 打破固定句式节奏 |
| 12 | 破折号过度使用 | 改用逗号、句号或另起一句 |
| 13 | 否定式排比——"不是A，不是B，而是C" | 直接说"是C" |
| 14 | 连接性短语泛滥——"不仅如此""更令人惊讶的是" | 删除，让内容自然衔接 |
| 15 | "仿佛""似乎"过度使用 | 直接陈述，减少模糊化 |

## 人设崩塌 / 声音趋同

| # | 陷阱 | 避免方式 |
|---|------|----------|
| 16 | 不同角色说话风格雷同 | 给每个角色定义独特的语言习惯 |
| 17 | 角色行为不符合既定性格 | 写动作前查人物设定 |
| 18 | 主角突然降智推动剧情 | 用合理的方式制造困境 |
| 19 | 反派脸谱化——为坏而坏 | 给反派合理的动机和逻辑 |
| 20 | 配角工具人——只在需要时出现 | 给配角独立的行为线 |
| 21 | 人物名字前后不一致 | 统一使用设定文件中的名字 |

## 节奏失控 / 过渡生硬

| # | 陷阱 | 避免方式 |
|---|------|----------|
| 22 | 开场信息轰炸——第一章塞入过多设定 | 设定分散到剧情中逐步揭示 |
| 23 | 平淡章节过长——连续几章无冲突 | 每章至少一个小冲突或转折 |
| 24 | 高潮后急转直下——高潮结束马上切日常 | 高潮后给读者情绪缓冲 |
| 25 | 机械降神——困境用突然的能力解决 | 提前埋下伏笔或能力来源 |
| 26 | 场景切换突兀——没有过渡就换场景 | 用环境/时间线索做衔接 |
| 27 | 对话推动剧情但无动作描写 | 对话配合动作、神态、心理 |

## 逻辑漏洞 / 伏笔不回收

| # | 陷阱 | 避免方式 |
|---|------|----------|
| 28 | 时间线矛盾——前后时间对不上 | 维护时间线文档 |
| 29 | 伏笔埋了但后面忘了回收 | 伏笔登记表，标注回收章节 |
| 30 | 世界观前后不一致——设定自相矛盾 | 维护设定文档，写前查阅 |
| 31 | 能力体系崩坏——战力忽高忽低 | 明确能力等级和限制条件 |
| 32 | 地理位置矛盾——A城到B城距离忽远忽近 | 维护地图和距离设定 |
| 33 | 角色关系错乱——A和B的关系前后矛盾 | 维护人物关系图 |

## 章节结构 / 悬念设计

| # | 陷阱 | 避免方式 |
|---|------|----------|
| 34 | 章节标题与内容不符 | 标题应暗示本章核心事件 |
| 35 | 章节末尾没有钩子——读者不想看下一章 | 每章结尾留悬念或转折 |
| 36 | 章节过长（>8000字）无分段感 | 大章分小节，用分隔线或空行 |
| 37 | 章节过短（<500字）信息量不足 | 合并到相邻章节或充实内容 |
| 38 | 开头重复上一章结尾 | 新章开头从不同角度或时间点切入 |
| 39 | 全章都是对话/全是描写，缺乏变化 | 对话、动作、心理交错使用 |
| 40 | 悬念揭晓太慢——铺垫过多但高潮平淡 | 控制铺垫与高潮的比例 |

## 题材特定

| # | 陷阱 | 避免方式 |
|---|------|----------|
| 41 | 修仙文：修炼等级体系混乱 | 明确等级名称和突破条件 |
| 42 | 修仙文：天材地宝命名随意 | 建立灵草/法宝命名规则 |
| 43 | 都市文：用现代词汇写古代背景 | 注意时代背景用词 |
| 44 | 穿越文：穿越者优势过度 | 合理限制穿越者的知识/能力 |
| 45 | 言情文：感情进展突兀 | 情感递进需要事件铺垫 |
| 46 | 系统文：系统功能忽有忽无 | 明确系统的功能和限制 |
| 47 | 悬疑文：线索太明显或太隐晦 | 线索应可被读者推理到 |
