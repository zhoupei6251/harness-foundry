# 小说审稿规则

## 自检清单

### 基础检查
- [ ] 无错别字和语病
- [ ] 标点符号规范（全角）
- [ ] 段落格式统一
- [ ] 章节标题与内容匹配

### 人设检查
- [ ] 角色行为符合设定
- [ ] 对话风格区分
- [ ] 名字前后一致
- [ ] 人物关系无矛盾

### 情节检查
- [ ] 情节推进合理
- [ ] 时间线无矛盾
- [ ] 伏笔有回收
- [ ] 无逻辑漏洞

### AI 痕迹检查
- [ ] 无 AI 高频词汇
- [ ] 无套路化表达
- [ ] 无机械过渡
- [ ] 朗读检查自然度

### 节奏检查
- [ ] 开头有吸引力
- [ ] 结尾有钩子
- [ ] 节奏张弛有度
- [ ] 无冗长段落

## 审稿流程
1. 通读全文，标记问题段落
2. 按严重程度排序（致命 > 高 > 中 > 低）
3. 给出具体修改建议
4. 评分（情节/人物/文笔/节奏 各维度）

## 评分标准
- 9-10 分：可直接发布
- 7-8 分：小幅修改后可发布
- 5-6 分：需要返修
- 3-4 分：需要大幅重写
- 1-2 分：建议重新构思
