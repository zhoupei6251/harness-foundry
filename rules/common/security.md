# 通用安全规则

## 敏感信息
- 禁止在代码中硬编码密码、API Key、Token
- 使用环境变量或配置中心管理密钥
- 日志中脱敏处理敏感数据
- 配置文件加入 .gitignore

## 输入验证
- 所有外部输入必须验证
- 使用白名单而非黑名单
- 验证类型、长度、范围、格式
- 服务端验证不可省略

## SQL 注入防护
- 使用参数化查询
- 禁止字符串拼接 SQL
- 使用 ORM 框架
- 定期扫描 SQL 注入漏洞

## XSS 防护
- 输出编码（HTML/JS/URL）
- 使用安全的模板引擎
- 设置 Content-Security-Policy
- 避免直接拼接用户输入

## 依赖安全
- 定期扫描依赖漏洞
- 锁定依赖版本
- 及时更新安全补丁
- 避免使用不维护的库
