# 新闻审稿规则

## 自检清单

### 事实核查
- [ ] 核心事实有两个以上信源
- [ ] 数据标注来源和统计方法
- [ ] 引语完整准确
- [ ] 时间地点人物准确

### 结构检查
- [ ] 标题准确概括内容
- [ ] 导语包含5W1H
- [ ] 信息按重要性排列
- [ ] 无冗余信息

### 合规检查
- [ ] 无诽谤和侵权内容
- [ ] 隐私信息已脱敏
- [ ] 版权内容已标注
- [ ] 敏感话题已审核

### 伦理检查
- [ ] 语言客观中立
- [ ] 无煽动性表述
- [ ] 多方观点平衡
- [ ] 无偏见和歧视

### 时效性检查
- [ ] 信息是最新的
- [ ] 标注发布日期
- [ ] 后续进展已更新
- [ ] 热点时效内发布

## 审稿流程
1. 事实核查：验证所有关键信息
2. 结构审查：检查新闻结构完整性
3. 合规审查：法律和伦理检查
4. 文字润色：语言和表达优化
5. 最终确认：发布前最后检查

## 评分标准
- 9-10 分：可直接发布
- 7-8 分：小幅修改后可发布
- 5-6 分：需要补充信息
- 3-4 分：需要重写部分段落
- 1-2 分：建议重新采写
