# 示例配置

展示不同项目如何接入 harness-foundry。

## 目录结构

```
examples/
├── README.md              # 本文件
├── code-project/          # 代码项目示例
│   └── project-structure.md
├── novel-project/         # 小说项目示例
│   └── project-structure.md
└── news-project/          # 新闻项目示例
    └── project-structure.md
```

## 使用方式

1. 参考对应类型的示例
2. 复制目录结构
3. 根据项目需求调整
4. 运行初始化脚本

## 示例列表

| 类型 | 示例 | 说明 |
|------|------|------|
| 代码项目 | `code-project/project-structure.md` | Java + Vue 全栈项目 |

| 新闻项目 | `news-project/project-structure.md` | 新闻采编项目 |
