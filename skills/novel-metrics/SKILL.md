# Novel Metrics — 写作指标追踪

> 统计和分析写作进度，提供量化反馈。
> 借鉴 ECC 的 metrics 追踪理念。

## 指标类型

### 1. 进度指标

```markdown
## 章节进度

| 指标 | 当前值 | 目标值 | 完成度 |
|------|--------|--------|--------|
| 已写章节 | 5 | 10 | 50% |
| 已写字数 | 16,500 | 30,000 | 55% |
| 预计完本 | 30 天 | - | - |
```

### 2. 速度指标

```markdown
## 写作速度

| 指标 | 数值 | 说明 |
|------|------|------|
| 本周均速 | 2,300 字/天 | 过去 7 天 |
| 昨日速度 | 3,200 字/天 | 昨天 |
| 最快单日 | 5,500 字/天 | 历史峰值 |
| 最慢单日 | 800 字/天 | 历史低谷 |

### 趋势
- 📈 较上周 +15%
- 目标速度：2,500 字/天
- 状态：✅ 超额完成
```

### 3. 质量指标

```markdown
## 审稿质量

| 指标 | 数值 | 说明 |
|------|------|------|
| 首次通过率 | 40% | 2/5 章一次通过 |
| 平均返修次数 | 1.6 次 | 返修后通过 |
| 最高评分 | 8.2 分 | 第 3 章 |
| 最低评分 | 6.8 分 | 第 1 章 |

### 趋势
- 评分趋势：📈 逐步提升
- 问题类型：AI 痕迹 > 节奏 > 人设
```

### 4. 活跃度指标

```markdown
## 写作活跃度

| 指标 | 数值 |
|------|------|
| 连续写作天数 | 5 天 |
| 本周活跃 | 4/7 天 |
| 本月总写作时间 | 12 小时 |
```

1. 章节进度

   - rework_count: 返修次数

## 目标设定

### 个人目标

```markdown
## 写作目标

### 短期目标（本周）
- 完成第 6-7 章
- 日均 2,500 字
- 首次通过率 > 50%

### 中期目标（本月）
- 完成第一卷（15 章）
- 总字数 > 45,000
- 平均评分 > 7.0

### 长期目标（本书）
- 完本 30 章
- 总字数 ~90,000
- 完本率 100%
```
