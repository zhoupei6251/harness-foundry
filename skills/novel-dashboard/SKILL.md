# Novel Dashboard — 小说进度仪表板

> 显示当前书籍的完整状态，包括章节进度、人物状态、伏笔状态。
> **当用户询问进度时自动触发。**

## 触发词

| 触发 | 说明 |
|------|------|
| 进度 | 查看写作进度 |
| 状态 | 查看当前状态 |
| 写了多少 | 查看章节列表 |
| 查看小说 | 显示仪表板 |

---

## 读取数据源

### 1. 从 MEMORY.md 读取

```markdown
文件：章节正文/MEMORY.md

读取内容：
- book_title: 书名
- chapter_index: 章节列表
- chapter_summaries: 章节摘要
- characters: 人物状态
- plot_devices: 伏笔状态
- status: 当前状态
```

### 2. 从大纲读取

```markdown
文件：章节正文/大纲.md

读取内容：
- planned_chapters: 计划章节
- current_arc: 当前幕
- next_arc: 下一幕
```

## 状态颜色

| 颜色 | 状态 | 说明 |
|------|------|------|
| 🟢 绿 | 稳定 | 状态正常，可继续 |
| 🟡 黄 | 警告 | 需要关注 |
| 🔴 红 | 危险 | 需要修复 |
| ⚪ 灰 | 未使用 | 尚未登场/未启用 |
| ✓ | 完成 | 已完成/已回收 |
| ○ | 进行中 | 进行中/已埋设 |
| × | 失败 | 未完成/未回收 |
