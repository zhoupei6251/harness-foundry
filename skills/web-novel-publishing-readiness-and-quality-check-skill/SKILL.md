# 小说质量检查技能

## 工具依赖
- **词频脚本**：`./freq_check.py`（必用，每章必跑）

## 核心原则

**禁用词优先级（五级铁律）：**
1. **禁用词优先** > 大纲原文 > 世界观设定
2. **词频扫描必须穷尽**整章，不得遗漏
3. **上下文穷尽原则**：同一词汇不同语境逐一判断
4. **主观→客观**：总结性描述替换为身体本能反应
5. **修正答案本身必须干净**：替换词不得含违禁词，不得改变原文语义/情感色彩

**动态高频词库**：每章检查完毕，将重复词（同一短语≥2次）追加入库，下次检查同步扫描。

**人写感优先**：文本读起来要像真实作者写的，而非AI批量生成。

## 执行要求：最大算力深度推理模式

检查必须完整执行五步推理链，不得省略跳步：
1. **任务全量解析**：核心需求/硬性约束/交付标准/问题边界
2. **多分支穷尽**：≥3条不同路径，逐一分析利弊，选最优+备选
3. **主路径全链路推理**：前提→执行→推导→校验，缺一不可
4. **多维交叉校验**：逻辑/需求匹配/严谨性/反向攻击
5. **最终交付物**：逻辑自洽+细节完整+推理依据+风险提示

**禁止偷懒**：禁止跳步/省略推理/占位符/TODO/套话凑数。

## 检查流程（十二步）

### 第一步：预检档位
| 档位 | 状态 | 操作 |
|:---|:---|:---|
| A档 | 段落错落，有对话有风格 | ✅保持 |
| B档 | 整体尚可，句式单一缺少停顿 | ⚠️改风险段 |
| C档 | 大段描述高度一致，像说明书 | 🔧必须改 |

### 第二步：违禁词扫描（必须穷尽，逐条列行号）

| 问题类型 | 修正方向 |
|---------|---------|
| 比喻词：如、像、如同、仿佛、好像 | 禁用 |
| 状态副词：猛然、陡然、骤然、缓缓 | 替换或具体动作 |
| 模板句式：冷笑一声、一阵剧痛、涌来、抖了抖、偏了偏 | 具体神态/体感/动词 |
| 感官：腥气、腥味、腥甜、颤抖、盯着、盯着看 | 其他感官词或具体动作 |
| 肢体动作类：没有回答、没有离开、站在原地、愣了、愣住了、愣了一下、抬起手、低头、看着手掌/手心/手背、没说话、没回、没应、看着她/他、盯着他/她、没动、没敢动、没有动、没回头、一沉、心里一沉 | 替换或删除 |
| 身体部位：指尖、指节、嘴角、嘴唇 | 手指、手骨等 |
| 生理反应：眼眶通红、绷得发白、哆嗦了半天 | 其他生理反应描写 |
| 水文痕迹：没人说话、整整 | 删除或具体细节 |
| 废话伏笔：他知道、她知道、可他不去找她她就不会来找他 | 删除 |
| X了X格式：攥了攥、偏了偏、收了收 | 动词替代 |
| 英文残留 | 中文替代 |

### 第三步：模板化句子
- **原则**：不是换词，是换画面/动作/场景
- 抽象心理→身体反应；固定句式→全新场景；模板比喻→原创画面
- **来源**：仅用字典基础词重组，不用AI库词汇
- 例："目光冷得像冰"（禁）→"他把脸别开"✓

### 第四步：词频检查（必跑freq_check.py）
- 任意词汇一章仅出现1次（人名除外）
- 四字成语全文禁止
- 词频脚本≠违禁词扫描，两步必须同执行

### 第五步：章节衔接
- 上章结尾→本章开头是否衔接
- ⚠️跨章节重复检查：调出上章结尾500字与本章开头500字逐段对比

### 第六步：大纲对照
- 正文是否偏离主线/关键情节点
- 角色行为是否与大纲设定一致

### 第七步：人设一致性
- 语言风格/关键特征/反应是否前后统一
- ⚠️ NPC必须正经名字，禁止"刀疤脸""火系异能者"等地摊称呼
- 发现一处地摊称呼→全文扫描所有同类，一次性列完

### 第八步：节奏代入感
- 对话比率/情绪起伏/代入感是否到位

### 第九步：重复内容
- 不同角色说出相同的话/相同动作场景重复出现

### 第十步：数量词术语一致性
- 等级/人数/距离/时间线是否矛盾

### 第十一步：世界观一致性
- 瞳色/技能/等级体系/势力名称是否统一

### 第十二步：复核自检（满足2/3项合格）
- 有无"依据/来源/伏笔"
- 有无"条件/边界/例外"
- 有无"取舍/代价/备选"

**修正后必须重跑第二步+第四步，确认无新增问题方可出报告。**

## 输出格式

```
🔍 【章节名】质量检查报告
📌 档位评估：[A档/B档/C档]
❌ 违禁词/套路句：[行号] "原文" → "修正"
⚠️ 模板化句子：[行号] "原文" → "替换方案"
🔗 章节衔接：[衔接/问题]
📋 大纲对照：[未偏离/偏离描述]
👤 人设一致性：[一致/问题]
🎭 节奏代入感：[达标/问题]
🔄 重复内容：[有/无+描述]
🔢 数量词术语：[一致/问题]
🌍 世界观一致性：[一致/问题]
✅ 复核自检：[满足X/3项]
📝 总体评价：[通过/需修改/重大问题]
```
