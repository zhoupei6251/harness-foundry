# Novel Init — 新书创作向导

> 帮助用户从零开始创建小说项目。
> **完整的 onboarding 流程：设定 → 大纲 → 人物 → 写作。**

## 触发词

| 触发 | 说明 |
|------|------|
| 写小说 | 创建新书 |
| 新书 | 新书向导 |
| 从头开始 | 重新开始 |
| 创建 | 创建新项目 |

---

## Step 1: 收集信息

### 快速模式（用户提供完整信息）

```markdown
用户: "写玄幻小说，主角穿越到修仙世界，预计50章"

直接进入：
1. 提取信息
2. 生成大纲
3. 创建项目结构
4. 等待确认
```

### 引导模式（信息不全）

```markdown
用户: "写小说"

┌─────────────────────────────────────────┐
│  📖 新书创作向导                          │
├─────────────────────────────────────────┤
│                                          │
│  请告诉我：                               │
│                                          │
│  1. 你想写什么题材的故事？                │
│     [玄幻] [都市] [科幻] [悬疑] [其他]   │
│                                          │
│  2. 大概是什么设定？                      │
│     [自由输入]                           │
│                                          │
└─────────────────────────────────────────┘
```

## Step 2: 生成设定

### 世界观生成

```markdown
基于用户输入，生成：

- 核心设定（1-2句话）
- 世界观特点（3-5个）
- 能力体系（如果有修仙等）
- 社会结构
- 时代背景
```

### 人物生成

```markdown
基于设定，自动生成：

主角：
- 姓名：
- 身份：
- 性格特点：
- 成长弧线：
- 金手指（如有）：

配角（2-3个）：
- 姓名/身份/性格/与主角关系
```

---

## Step 3: 创建大纲

### 大纲模板

```markdown
# 《{书名}》大纲

## 基本信息
- 题材：{题材}
- 规模：{预计章数}
- 核心卖点：{1-2句话}

## 世界观设定
{详细世界观}

## 主要人物
{人物表}

## 主线剧情

### 第一幕（铺垫）
- 章节：1-{N}
- 目标：{目标}
- 关键事件：{事件列表}

### 第二幕（对抗）
- 章节：{N+1}-{M}
- 目标：{目标}
- 关键事件：{事件列表}

### 第三幕（解决）
- 章节：{M+1}-{end}
- 目标：{目标}
- 关键事件：{事件列表}

## 伏笔安排
{伏笔列表}

## 章节规划
| 章节 | 主要内容 | 目标 |
|------|----------|------|
| 第1章 | ... | ... |
| 第2章 | ... | ... |
```

## Step 4: 创建项目结构

### 目录结构

```
章节正文/
├── 《{书名}》/
│   ├── MEMORY.md           # 记忆文件
│   ├── 大纲.md              # 完整大纲
│   ├── 人物设定/            # 人物详细设定
│   │   ├── 主角.md
│   │   └── 配角.md
│   ├── 世界观/              # 世界观设定
│   │   ├── 基础设定.md
│   │   └── 能力体系.md
│   └── 第X章_{标题}.md     # 各章正文
│
└── .novel-workspace/       # 工作区元数据
    ├── config.yaml
    └── backup/             # 自动备份
```

### 核心文件

#### MEMORY.md 模板

```markdown
# 《{书名}》记忆文件

> 自动更新，维护跨会话上下文

- 创建时间：{日期}
```

## 交互式问答

### 最小信息集

```markdown
必须收集：
□ 书名
□ 题材
□ 主角设定

可选（可自动生成）：
□ 世界观
□ 配角
□ 详细大纲
```

### 问答示例

```
用户: "写修仙小说"

AI: 请告诉我：
    1. 书名？（可以后取）
    2. 主角身份？（废物流/世家子/穿越者...）
    3. 核心冲突？（打怪升级/夺宝/复仇...）

用户: "废物流，主角从废物逆袭"

AI: ✓ 设定已提取
    书名：《逆天改命》
    题材：玄幻修仙
    主角：废物逆袭型

    [确认设定]  [调整主角]  [继续]
```
