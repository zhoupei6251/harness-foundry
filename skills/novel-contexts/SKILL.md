# Novel Contexts — 上下文一致性管理

> 维护小说创作中的全局一致性：角色设定、世界观、时间线、能力体系。
> **这是 novel 域的"类型系统"——防止不一致的源头。**

## 核心概念

### 上下文层级

```markdown
┌─────────────────────────────────────────┐
│  📖 全局上下文（对全书有效）             │
│  ├── 世界观设定                         │
│  ├── 能力体系                           │
│  ├── 时间线                              │
│  └── 核心人物关系                        │
├─────────────────────────────────────────┤
│  📝 卷/幕上下文（对当前卷有效）         │
│  ├── 当前卷的特殊规则                   │
│  ├── 本卷主要冲突                       │
│  └── 重要NPC                           │
├─────────────────────────────────────────┤
│  📄 章节上下文（对当前章有效）           │
│  ├── 场景设定                           │
│  ├── 当前目标                           │
│  └── 涉及角色                           │
└─────────────────────────────────────────┘
```

## 上下文检查

### 1. 人物一致性检查

```markdown
## 人物上下文

检查项：
□ 姓名：{前后一致}
□ 外貌：{描述是否冲突}
□ 性格：{行为是否符合}
□ 能力：{不超出设定范围}
□ 关系：{与其他角色关系}

常见冲突：
- 开头设定"内向"，后文"滔滔不绝"
- 开头"剑术高手"，后文"被普通人打败"
```

### 2. 世界观一致性检查

```markdown
## 世界观上下文

检查项：
□ 地理：{地名是否统一}
□ 势力：{门派/组织设定是否一致}
□ 规则：{超自然规则是否自洽}
□ 科技：{技术水平是否矛盾}

常见冲突：
- 古代背景出现现代词汇
- 低武世界出现毁天灭地的能力
```

### 3. 时间线检查

```markdown
## 时间线上下文

检查项：
□ 年份：{朝代/纪年是否一致}
□ 季节：{是否符合情节逻辑}
□ 时长：{修炼/路程所需时间}

常见冲突：
- A 说"三年后"，B 说"五年后"
- "上午还在北京" → "傍晚到广州"（不可能）
```

### 4. 能力体系检查

```markdown
## 能力上下文

检查项：
□ 等级：{当前等级是否正确}
□ 能力：{使用的技能是否已解锁}
□ 消耗：{能力代价是否一致}
□ 冷却：{CD 时间是否矛盾}

常见冲突：
- 还没学的能力已经使用
- 上章"内力耗尽"，下章"全力一击"
```

## 上下文文件

### 世界观文件

```markdown
# 世界观.md

## 基础设定
- 世界名称：{名称}
- 时代背景：{朝代/纪年}
- 地理格局：{主要势力分布}

## 能力体系
- 修炼等级：{等级名称列表}
- 能力限制：{限制条件}
- 特殊能力：{独特能力来源}

## 社会结构
- 主要势力：{列表}
- 势力关系：{关系图}
```

- 口头禅：
- 行为模式：

## 能力
- 当前等级：
- 技能列表：
- 限制/弱点：

## 关系
- 盟友：
- 敌人：

### 时间线

```markdown
# 时间线.md

| 第一年3月 | 拜师 | 第3章 |
| 第一年6月 | 第一次战斗 | 第5章 |
```

## 一致性验证流程

```markdown
写章节前的检查流程：

1. 加载全局上下文
   - 读取 世界观.md
   - 读取 人物档案/
   - 读取 时间线.md

2. 提取本章需要的上下文
   - 本章涉及的角色
   - 本章涉及的地点
   - 本章的时间点

3. 检查冲突
   - 人物能力是否超出
   - 地点描述是否矛盾
   - 时间逻辑是否成立

4. 报告潜在问题
   - 如有问题，在写作前解决
```

## 冲突处理

### 检测到冲突时的处理

```markdown
⚠️ 检测到上下文冲突

冲突类型：人物能力矛盾
涉及：主角 {姓名}

冲突描述：
- 设定：当前筑基期（第8章）
- 本章使用：结丹期技能（第12章）

解决方案：
A. 修改本章，使用筑基期能力
B. 修改时间线，在中间插入突破
C. 添加剧情解释（如秘法暂时提升）

请选择 [A/B/C]：
```

## 禁止事项

- ❌ 忽略上下文冲突继续写
- ❌ 临时添加设定不记录
- ❌ 随意改变人物能力
- ❌ 忽视时间线矛盾
