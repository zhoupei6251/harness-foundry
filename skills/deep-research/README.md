# Deep Research - 深度研究技能

## 简介

Deep Research 是一个专为 AI Agent 设计的深度研究技能。每次用户提问时，它自动执行以下流程：

1. **本地知识库搜索** - 搜索工作空间内的所有知识库文件
2. **互联网多平台搜索** - 从多个互联网平台搜索相关内容
3. **整合分析** - 结合搜索信息和自身认知进行深度分析
4. **完整推送** - 生成结构完整、内容详实的回答

## 核心原则

### 深度优先于速度
宁可花更多时间获取完整信息，也不推送不完整的内容。

### 完整性优先于简洁
提供结构完整的内容：背景 → 分析 → 结论 → 建议，包括数据来源和引用。

### 问题优先解决
遇到问题立即解决，而不是立即换其他方案。每个问题至少尝试3种不同的解决方案。

## 工作流程

### 7大标准步骤

| 步骤 | 名称 | 核心内容 |
|------|------|----------|
| 1 | 接收用户问题 | 理解核心需求，识别关键概念 |
| 2 | 本地知识库搜索 | 搜索知识库文件，提取相关信息 |
| 3 | 互联网多平台搜索 | 网页搜索 + 多平台搜索 |
| 4 | 整合分析 | 分类整理、去重合并、可信度评估 |
| 5 | 生成完整回答 | 标准结构输出 |
| 6 | 问题处理机制 | 问题识别→解决路径→禁止行为 |
| 7 | 持续改进 | 反馈收集→知识更新→技能优化 |

### 回答标准结构

```
## [标题]
### 摘要
### 背景分析
### 核心内容
### 深度洞察
### 实用建议
### 参考资料
### 补充说明
```

## 搜索覆盖范围

### 本地知识库
- 产品知识库
- 销售话术库
- 销售技巧库
- 产品推荐系统
- 培训材料索引
- 保险法律法规知识库
- 常见问题
- 查询索引

### 互联网平台
- 通用网页搜索
- 网页内容抓取
- 多平台搜索（需配合07技能）

## 使用示例

### 产品咨询
**用户**：泰盈人生2026庆典版适合什么人买？
**执行**：搜索知识库 → 搜索互联网评价 → 整合分析 → 推送完整报告

### 行业趋势
**用户**：当前保险行业的最新趋势是什么？
**执行**：搜索知识库 → 搜索行业报告 → 整合分析 → 推送趋势分析

### 比较分析
**用户**：泰康和其他保险公司相比有什么优势？
**执行**：搜索知识库 → 搜索竞品分析 → 多维度对比 → 推送对比报告
