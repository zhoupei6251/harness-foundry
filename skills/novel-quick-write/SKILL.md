# Novel Quick Write — 快速单章写作

> 当用户说"写第X章"且无复杂需求时使用此模式。
> **跳过完整编排流程，直接写章节。**

## 触发条件

- 用户说"写第X章"
- 用户说"续写"
- 无需审稿/润色的简单写作
- 当前书籍已有 MEMORY.md

## 快速流程

```
1. 读取 MEMORY.md
       ↓
2. 加载上下文（人物状态/伏笔/上章摘要）
       ↓
3. 写章节（≥2000字）
       ↓
4. 自检（字数 + AI痕迹）
       ↓
5. 交付
```

---

## 详细步骤

### Step 1: 读取 MEMORY

```markdown
读取文件：章节正文/MEMORY.md

获取：
- 当前书籍名称
- 已完成章节列表
- 人物状态
- 伏笔状态
- 上章摘要
```

### Step 2: 加载上下文

```markdown
从 MEMORY.md 提取：
- 上一章（第X-1章）结尾
- 本章目标
- 涉及人物
- 伏笔安排
```

### Step 3: 写作

```markdown
写章节：第{XXX}章_{标题}.md

要求：
- 承接上一章结尾
- 完成本章目标
- 人物对话符合设定
- ≥2000字（包含标点）
- 全角标点
- 结尾留钩子
```

### Step 4: 自检

```python

    return len(text)

```

### Step 5: 更新 MEMORY

```markdown
更新章节正文/MEMORY.md：
- 添加本章摘要
- 更新人物状态（如有变更）
- 更新伏笔状态（如有新增/回收）
- 更新完成章节列表
```
