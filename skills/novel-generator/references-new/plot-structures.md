# 爽文情节结构参考

经典的爽文情节套路和节奏设计参考。

## 经典爽文结构公式

### 单章节奏（微观）

```
铺垫（前20%）→ 冲突升级（中间50%）→ 爆发/反转（后20%）→ 钩子（最后10%）
```

### 卷级节奏（中观）

```
第1章：困境建立 + 获得机缘
第2-3章：小试牛刀 + 第一次打脸
第4-5章：实力提升 + 引来更强对手
第6-8章：连番挑战 + 主角成长
第9-10章：卷级 BOSS 战 + 大高潮
第11章：新篇开启 / 更高层次世界
```

### 全书节奏（宏观）

```
开篇卷（1-10章）：废柴逆袭，从底层崛起
发展卷（11-30章）：扬名立万，碾压同辈
转折卷（31-50章）：遭遇挫折，绝地反击
高潮卷（51-70章）：巅峰对决，真相揭示
终章卷（71-80章）：终极一战，功成名就
```

## 十大经典爽文套路

### 1. 扮猪吃虎

```
伪装弱者 → 被人看不起 → 关键时刻展示实力 → 众人震惊
```

**适用频率**：每 3-5 章一次
**变体**：装穷、装傻、隐藏修为、故意示弱

### 2. 退婚打脸

```
被退婚/被抛弃 → 发奋图强/获得机缘 → 实力飞涨 → 前任/对方后悔莫及
```

**适用频率**：通常作为开篇引擎
**变体**：被逐出家族、被开除宗门、被解除婚约

### 3. 拍卖会装逼

```
进入拍卖场 → 竞价被人嘲笑没钱 → 一掷千金碾压 → 众人震惊身份
```

**适用频率**：每 10-15 章一次
**变体**：赌石、炼丹大会、比武招亲

### 4. 越级挑战

```
面对高等级对手 → 众人认为必败 → 出人意料获胜 → 声名大噪
```

**适用频率**：每 5-8 章一次
**变体**：以一敌多、以低境界胜高境界、不用全力就获胜

### 5. 身份揭秘

```
以普通身份行事 → 遭遇困难/被刁难 → 真实身份曝光 → 所有人跪了
```

**适用频率**：每 15-20 章一次（不宜太频繁）
**变体**：师承揭秘、血脉觉醒、势力背景曝光

### 6. 绝境突破

```
陷入绝境 → 生死之间感悟 → 突破境界 → 反杀
```

**适用频率**：每 8-12 章一次
**变体**：战斗中突破、服用丹药突破、机缘巧合

### 7. 打脸势利眼

```
势利眼狗眼看人低 → 主角展示实力/身份/财力 → 势利眼跪舔 → 主角冷漠离去
```

**适用频率**：每 3-5 章穿插
**变体**：药店、武器店、宗门入门

### 8. 故人重逢

```
早期看不起主角的人 → 多年后再次相遇 → 发现主角已飞黄腾达 → 后悔/震惊/跪
```

**适用频率**：每 20-30 章一次
**变体**：同学聚会、回乡、宗门大会

### 9. 逆转局势

```
局势一边倒（不利于主角方）→ 主角出场/使出底牌 → 一己之力扭转 → 全场欢呼
```

**适用频率**：卷终高潮必用
**变体**：战场逆转、谈判逆转、比赛逆转

### 10. 连环打脸

```
打脸 A → A 叫来更强的 B → 打脸 B → B 叫来更强的 C → 打脸 C → 最后大 BOSS 出面 → 终极碾压
```

**适用频率**：大高潮章节
**变体**：层层递进式、一气呵成式

## 情绪曲线设计

### 低开高走（最常用）

```

  │            ╱──── 爽点爆发
  │          ╱
  │        ╱
  │      ╱ 
  │    ╱   渐入佳境
  │──╱ 
  │╱  开局低谷
  └─────────────── 章节进度
```

```
情绪值
  │   ╱╲      ╱╲

  │ ╱    ╲  ╱    ╳

  │          ╲  ╱
  │           ╲╱ ← 最低点 = 转折点
  └─────────────── 章节进度
```

## 伏笔设计规范

### 伏笔类型

| 类型 | 埋设 | 回收间隔 | 示例 |
|------|------|---------|------|
| 短期伏笔 | 本章暗示 | 1-3 章后回收 | 神秘人的身份 |
| 中期伏笔 | 描写细节 | 5-15 章后回收 | 主角身上的奇异胎记 |
| 长期伏笔 | 不经意提及 | 20+ 章后回收 | 开篇提到的一句预言 |

### 伏笔检查

每 5 章检查一次 `PLOT_POINTS.md`，确保：
- 状态为"伏笔待回收"的条目不超过 5 个
- 没有超过 30 章仍未回收的伏笔
- 已回收的伏笔标记为"已完结"

## 章节衔接技巧

### 好的衔接

- 承接上章钩子，但不完全重复
- 视角切换（从主角切到对手的反应）
- 时间跳跃（"三天后..."）+ 简述间隔期变化

### 差的衔接（避免）

- 完全重复上章最后一段
- 毫无关联地切换场景
- 大段回忆/解释打断节奏
